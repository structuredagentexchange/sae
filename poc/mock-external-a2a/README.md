# Mock External A2A POC

This folder holds a draft external-facing A2A-shaped agent artifact for SAE experiments.

- this is still a mock external participant
- it is meant to demonstrate the adoption shape
- it is not yet proof of a third-party implementation

Use it to show:

- how an outside agent could declare SAE support
- how a narrow external agent might advertise its skills
- how SAE support can be framed without requiring product-specific language
